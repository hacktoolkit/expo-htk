# Hacktoolkit HTK Documentation

Welcome to the **Hacktoolkit HTK** comprehensive documentation! This is a React Native/Expo application toolkit providing reusable components, utilities, and features for building mobile applications.

## 🚀 Quick Navigation

### Core Modules

- **[Components](components/)** - Reusable React Native UI components
- **[Features](features/)** - Feature-specific implementations and integrations
- **[Utilities](utils/)** - Helper functions and utilities
- **[Storages](storages/)** - Data persistence and storage adapters
- **[States](states/)** - State management patterns
- **[Types](types/)** - TypeScript type definitions

## 📚 What's Inside

### Components
Pre-built, production-ready React Native components including dialogs, confirmations, and more.

### Features
Complete feature implementations including:
- App Settings management
- Theme system (light/dark mode)
- Device information access
- Geolocation utilities

### Utilities
Helper utilities for common tasks:
- String manipulation and formatting
- Geolocation calculations
- Number humanization
- Enum utilities
- React Context builders
- Observer pattern implementation
- Theme styling helpers

### Storages
Data persistence solutions:
- MMKV (Memory-Mapped Key-Value) storage
- localStorage adapter
- Storage abstraction layer

### States
State management with Jotai atoms:
- Reactive state updates
- Automatic persistence
- Suspense integration
